# SVG Review Assets Index

Use this directory together with `references/svg-review/`.

## What Lives Here

- `review_log_template.md` — page-by-page and whole-deck review record template
- `fix_task_template.md` — turn findings into executable fix work
- `user_confirmation_template.md` — export approval template before post-processing

## Recommended Usage Order

1. Review each generated page and log it with `review_log_template.md`
2. When a real issue is found, track it with `fix_task_template.md`
3. After the whole deck passes review, prepare user approval with `user_confirmation_template.md`

## Responsibility Split

- `references/svg-review/` defines the rules
- `templates/review/` provides the execution formats
