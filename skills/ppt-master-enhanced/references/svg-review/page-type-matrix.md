# SVG Page-type Matrix

## 1. How to Use

When a page issue appears:

1. identify the page type first
2. look up the typical symptom
3. apply the preferred fix
4. confirm the result in the review log

## 2. Page-type Lookup

| Page Type | Typical Symptoms | Preferred Fix | Review Focus |
|----------|------------------|---------------|--------------|
| TOC / Agenda | explanation block crowds the bottom; left/right blocks drift | shorten explanation, add padding or height, align to shared top line | explanation no longer touches bottom; left/right blocks feel on one grid |
| Table | first column squeezes second column; header and body do not share column logic | recalculate widths, gaps, anchors; make first column left-aligned if needed | first column is readable; header/body stay vertically aligned |
| Quadrant / Matrix | items in the same quadrant do not share a line; left and right sides feel uneven | rebuild the reference lines and move as a group | shared baseline and balanced composition |
| Process / Flow | whole card set drifts; arrows enter next card; extra arrow remains after last step | rebuild centered grid; stop arrows before target; remove extra terminal arrow | flow reads clearly and connectors stay in spacing zones |
| Architecture | summary band hits body area; connectors cover center node; center area is too dense | increase band spacing; draw connectors before focal node; reduce center copy density | center node stays visually on top and body bands are separated |
| List / Value Proposition | small text crowds the background strip; row rhythm is inconsistent | add strip height or padding; normalize each row structure | rows keep a stable rhythm and safe margins |
| Three-phase / Roadmap | three columns do not share an axis; tags/arrows/cards do not align | rebuild three-column grid; unify center lines and list baselines | three columns read as one system |
| Cover / Closing | title crowds edge; subtitle is too long; decoration steals focus | shorten subtitle, increase title safe margin, reduce decoration contrast | first glance lands on the title |

## 3. Symptom Reverse Lookup

| Symptom | Check First | Preferred Fix |
|--------|-------------|---------------|
| Overflow | copy length, container width, anchor | shorten copy, split lines, resize container |
| Edge crowding | padding, line count, container height | add padding, increase height, reduce line count |
| Misalignment | top line, center line, column width, gap | rebuild grid and use shared references |
| Covered arrows or nodes | end point, draw order | shorten connector end, reorder layers |
| Unstable visual balance | information density, visual emphasis | reduce secondary density, strengthen hierarchy |
